# <mark style="background: #ABF7F7A6;">Daily Notes</mark>

Use `dotnet new gitignore` to generate a gitignore template for unnecessary dotnet files

It is also a good idea to add `appsettings.json` to the gitignore list as it can contain private keys

## More [[Vs Code Shortcuts]]:
- In vscode use `ctor` to generate a snippet for the constructor
- Use Ctrl + . to trigger quick actions in vscode on a variable, which will let you generate private fields for a constructor parameter

## More [[Obsidian Syntax]]:
- Add `^reference` to the end of a note to add a reference to the block
- Reference the block later with `![[name-of-note#^reference]]`
